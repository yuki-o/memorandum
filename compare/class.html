## ~ クラス ~ ### C言語 * なし --- ### C++ 1. ヘッダに多重インクルードガードが必要. 1. クラス宣言の } 直後のセミコロンを忘れずに。 2. アクセス修飾子は範囲毎に指定する。 3. アクセサは自動生成されないので、他のメンバメソッドと同様に宣言する必要がある。 4. コンストラクタでのメンバ変数の初期化は、コンストラクタ初期化子 : を用いて行う。 5. インスタンスを、スタック領域に確保する方法と、ヒープ領域に確保する方法がある。 __triangle.h__ ```cpp // (1). #ifndef __triangle__ #define __triangle__ class Triangle { // (3). private: float base; // 底辺. float height; // 高さ. // (3). public: // コンストラクタ. Triangle( float i_fBase, float i_fHeight ); // (4) baseのアクセサ. float getBase() const { return base; } void setBase( float i_fBase ) { base = i_fBase; } // (4) heightのアクセサ. float getHeight() const { return height; } void setHeight( float i_fHeight ) { height = i_fHeight; } // 面積を取得. float getArea() const; }; // (2). // (1). #endif /* defined(__triangle__) */ ``` __triangle.cpp__ ```cpp #include "triangle.h" // (5). Triangle::Triangle( float i_fBase, float i_fHeight ) : base( i_fBase ) , height( i_fHeight ) { } float Triangle::getArea() const { return getBase() * getHeight() * 0.5f; } ``` __main.cpp__ ```cpp #include <iostream> #include "triangle.h" using namespace std; int main( int argc, const char * argv[] ) { // (6) スタック領域に確保 (.演算子でメンバにアクセス). Triangle t1( 10.0f, 20.0f ); cout << "底辺:" << t1.getBase() << ", 高さ:" << t1.getHeight() << ", 面積:" << t1.getArea() << endl; // (6) ヒープ領域に確保 (->演算子でメンバにアクセス). Triangle* t2 = new Triangle( 10.0f, 20.0f ); cout << "底辺:" << t2->getBase() << ", 高さ:" << t2->getHeight() << ", 面積:" << t2->getArea() << endl; delete t2; return 0; } ``` --- ### Java 1. アクセス修飾子はそれぞれのメンバに個別で指定する。 2. アクセサは自動生成されないので、他のメンバメソッドと同様に宣言する必要がある。 ただし、Eclipseには自動生成してくれる機能が備わっているので、それを使用した方がラク。 3. main関数のアクセス修飾子は必ず public でなければならない。 4. 1つのファイルに、publicなクラスを1つしか定義できない。 5. クラス名とファイル名を同じにしなければならない。 __Triangle.java__ ```java public class Triangle { // (1). private float base; // 底辺. private float height; // 高さ. // コンストラクタ. public Triangle( float base, float height ) { setBase( base ); setHeight( height ); } // (2) baseのアクセサ. public float getBase() { return base; } public void setBase( float base ) { this.base = base; } // (2) heightのアクセサ. public float getHeight() { return height; } public void setHeight( float height ) { this.height = height; } // 面積を取得. public float getArea() { return getBase() * getHeight() * 0.5f; } } ``` __Main.java__ ```java public class Main { // (3). public static void main( String[] args ) { Triangle t = new Triangle( 10.0f, 20.0f ); System.out.println( "底辺:" + t.getBase() + ", 高さ:" + t.getHeight() + ", 面積:" + t.getArea() ); } } ``` --- ### C# # 1. アクセス修飾子はそれぞれのメンバに個別で指定する。 2. プロパティにより、アクセサを自動生成できる。 __Triangle.cs__ ```csharp class Triangle { // (1)(2) プロパティ. public float Base { set; get; } // 底辺. public float Height { set; get; } // 高さ. // コンストラクタ. public Triangle( float i_fBase, float i_fHeight ) { Base = i_fBase; Height = i_fHeight; } // 面積を取得. public float GetArea() { return Base * Height * 0.5f; } } ``` __Program.cs__ ```csharp using System; class Program { static void Main( string[] args ) { var t = new Triangle( 10.0f, 20.0f ); Console.WriteLine( "底辺:" + t.Base + ", 高さ:" + t.Height + ", 面積:" + t.GetArea() ); } } ``` --- ### Objective-C 1. NSObject を継承する必要がある。 2. アクセス修飾子は範囲毎に指定する。 3. プロパティにより、アクセサを自動生成できる。 クラス内では、メンバ変数名の前にアンダースコアをつけることでアクセス可能 (3')。 クラス外では、そのままのメンバ変数名でアクセス可能だが ->演算子 ではなく .演算子 を使用する必要がある (3'')。 4. @interface 内にメンバメソッドの実装を記述することができない。メンバメソッドの実装は @implementation 内で行う。 __triangle.h__ ```objectivec #import <Foundation/Foundation.h> @interface Triangle : NSObject // (1). { // (2). @private int test; } // (3) プロパティ. @property float base; // 底辺. @property float height; // 高さ. // (4) イニシャライザ. -(id) initWithBase:(float)i_fBase Height:(float)i_fHeight; // 面積を取得. -(float) getArea; @end ``` __triangle.m__ ```objectivec #import "triangle.h" @implementation Triangle // (4). -(id) initWithBase:(float)i_fBase Height:(float)i_fHeight { if( self = [super init] ) // 基底クラスの初期化に成功したとき. { // (3'). _base = i_fBase; _height = i_fHeight; } return self; } -(float) getArea; { return _base * _height * 0.5f; } @end ``` __main.m__ ```objectivec #import <Foundation/Foundation.h> #import "triangle.h" int main( int argc, const char * argv[] ) { @autoreleasepool { Triangle* t = [[Triangle alloc] initWithBase:10.0f Height:20.0f]; NSLog( @"底辺:%.1f, 高さ:%.1f, 面積:%.1f", t.base, t.height, [t getArea] ); // (3''). } return 0; } ``` --- ### PHP TODO ```php $a = 10; ``` --- ### JavaScript TODO ```js var a = 10; ``` --- [戻る](../index.html) --- &copy; 2014 小川優樹 all rights reserved.
